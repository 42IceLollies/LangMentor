Production Dependencies:

pg - For PostgreSQL database management
cors - For handling Cross Origin Resource Sharing (cors)
jsonwebtoken - For authentication with JWTs
dotenv - For creating secure environment variables
express - For creating a server for the backend
bcryptjs - For hashing passwords for the database
nodemailer - For sending verification emails


Development Dependencies:

nodemon - For automatically restarting the server on changes


Dependencies Install Command:

npm install pg cors jsonwebtoken dotenv express bcryptjs nodemailer --save
npm install nodemon --save-dev
